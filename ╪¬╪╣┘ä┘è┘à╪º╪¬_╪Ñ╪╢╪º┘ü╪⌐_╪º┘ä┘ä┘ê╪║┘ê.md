# 📌 تعليمات إضافة لوغو الرقيب القابضة

## نظرة عامة
تمّ تجهيز النظام بآلية مركزية لإضافة اللوغو. **مكان واحد فقط** تحتاج تعديله، ويظهر اللوغو تلقائياً في:
- الشريط العلوي (بجانب اسم النظام)
- شاشة تسجيل الدخول (في الأعلى)

---

## الخطوات

### افتح ملف `index.html` وابحث عن السطر التالي (قرب السطر 939):

```javascript
const COMPANY_LOGO_SRC = ''; // ← ضع هنا رابط اللوغو أو data URI
```

### لديك طريقتان للإضافة:

---

### ✅ الطريقة الأولى (الأسهل): ملف صورة بجانب index.html

1. ضع ملف اللوغو (مثلاً `logo.png`) في **نفس مجلد** `index.html`
2. عدّل السطر ليصبح:

```javascript
const COMPANY_LOGO_SRC = 'logo.png';
```

3. احفظ الملف وارفعه. انتهى.

> إذا كان اللوغو داخل مجلد فرعي مثل `assets`:
> ```javascript
> const COMPANY_LOGO_SRC = 'assets/logo.png';
> ```

---

### ✅ الطريقة الثانية (الأفضل للنشر): صورة مضمّنة (data URI)

تُضمّن الصورة داخل الملف نفسه — لا حاجة لملف منفصل، ولا تنكسر أبداً.

1. اذهب إلى موقع تحويل مجاني مثل: **https://www.base64-image.de/**
2. ارفع ملف اللوغو → انسخ النتيجة (تبدأ بـ `data:image/png;base64,...`)
3. الصقها بين علامتَي الاقتباس:

```javascript
const COMPANY_LOGO_SRC = 'data:image/png;base64,iVBORw0KGgoAAAANSUhEUg...';
```

4. احفظ وارفع. انتهى.

---

## ملاحظات مهمة

| النقطة | التفصيل |
|--------|---------|
| **صيغة اللوغو** | PNG (يفضّل بخلفية شفافة) أو SVG أو JPG |
| **الأبعاد المثالية** | مربّع تقريباً (مثل 200×200 أو 512×512) — النظام يضبط الحجم تلقائياً |
| **الشعار الافتراضي** | إذا تركت القيمة فارغة `''` يبقى الشعار الحالي (الأيقونة) |
| **لا تعدّل** | أي شيء آخر — فقط قيمة `COMPANY_LOGO_SRC` |

---

## التحقق بعد الإضافة
افتح النظام في المتصفّح:
1. **شاشة الدخول:** يجب أن يظهر اللوغو كبيراً في الأعلى
2. **بعد الدخول:** يجب أن يظهر اللوغو صغيراً في الشريط العلوي بجانب "نظام إدارة المعاملات والمهام"

إذا لم يظهر: تأكّد أن مسار الملف صحيح، أو أن نصّ الـ base64 لُصق كاملاً بين علامتَي الاقتباس.
